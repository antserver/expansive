Embedthis Expansive™ Overview

Expansive is a blazing fast and compact embedded web server for the efficient and secure hosting of embedded web applications. Expansive dramatically cuts the time and cost of developing web applications by including the ESP web framework and an extensive set of features.

Highly Optimized

Expansive is optimized for hosting dynamic embedded web applications via an event driven, multi-threaded core to deliver exceptional throughput and effective memory utilization. It is compact and will embed using as little as 1MB of memory (typical 2MB). It is especially effective in reducing the per-request memory overhead for web applications.

Extensive Features

Expansive has a strong set of features including: SSL/TLS, basic and digest authentication, virtual hosting, loadable modules, Apache style configuration file, CGI, ESP, PHP, sandbox resource limits, logging, request tracing, monitoring, defensive counter-measures, service manager, and extensive conditional configuration and compilation controls.

With these features, Expansive is designed to make hosting and embedding web applications painless. Expansive is the perfect server to host your web application.

Deployed Widely

As one of the most widely deployed embedded web servers, Expansive is being used in networking equipment, telephony, mobile devices, consumer and office equipment as well as hosting for enterprise web applications and frameworks. It runs equally well stand-alone or in a farm behind a reverse proxy such as Apache. Expansive provides both Open-Source and commercial licenses and comes with full source code.

Components

Expansive provides the following components:

  • HTTP web server program and library
  • HTTP client program and library
  • Manager and monitor processes
  • ESP web framework
  • Loadable modules for CGI, Ejscript, ESP, and PHP
  • Secure Socket Layer (SSL/TLS)
  • Support for other web frameworks: PHP and Ejscript
  • Complete documentation with samples
  • Full source code
