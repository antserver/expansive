Installing on Linux

Running Expansive on Linux is supported by building from source code.

Please see instructions at Building from Source.

Running Expansive

Expansive will start when the Expansive installation procedure completes and on some systems, will launch your browser directed at the Expansive local home page.Thereafter, Expansive will automatically start if the system is rebooted.

If you wish to manually stop Expansive, type:

appman start

To restart Expansive:

appman stop

Alternatively, you can run exp manually:

cd /etc/exp
exp

Expansive is configured by default to listen on port 80. To test that it is serving pages, point your browser at http://localhost. To tailor the Embedthis Expansive configuration, including the default port, you may wish to edit the exp.conf configuration file.

To get all the options for Expansive, read the man page via:

man exp
or get the command help via: 
exp --help
