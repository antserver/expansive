Installing on MAC OS X

The Embedthis Expansive install distribution for Mac OS X is published as a an installable pkg file.

Installing the PKG Image

  1. Double click on the PKG image.

Removing Expansive

  1. Run a terminal and change directory to the installation directory for Expansive.
  2. Run the remove script in the install directory for Expansive by typing:
sudo ./uninstall

Running Expansive

Expansive will start when the installation procedure completes. Thereafter, Expansive will automatically start when the system is rebooted.

If you wish to manually stop Expansive type:

sudo appman stop

To restart Expansive:

sudo appman start

Alternatively, you can run exp manually (if not already running):

cd /etc/exp
exp

Expansive is configured by default to listen on port 80. To test that it is serving pages, point your browser at http://localhost. To tailor the Embedthis Expansive configuration, including the default port, you may wish to edit the exp.conf configuration file.

To get all the options for Expansive read the man page via:

man exp
or get the command help via: 
exp --help
