Installing on Windows

The Embedthis Expansive install package for Windows is an installation executable compressed in a ZIP file called:

exp-VERSION-BUILD-MS-WIN-i686.exe.zip

The distribution image contains the binary distribution, documentation and development headers, and libraries in the one package. During installation you will have the option of installing the full package or portions of the distribution.

Installing the EXE Image

  1. Login with administrator privileges.
  2. Run winzip and extract the contents of the zipped EXE install image to a folder of your choosing.
  3. Browse to the location holding the EXE install image and run it.

Removing Expansive

  1. Login with administrator privileges.
  2. Run the Add / Remove Programs applet from the control panel.
  3. Select Embedthis Expansive and choose remove.

Silent Installation

If you need to do batch or unattended installations, you can install Expansive silently. Use the /silent switch to suppress questions and /verysilent to suppress all visual output.

Running Expansive

Expansive will start when the installation procedure completes and will launch your browser directed at the Expansive local home page. Thereafter, Expansive will automatically start if the system is rebooted.

If you wish to manually stop Expansive, right click on the Expansive Monitor icon in the system tray (lower left corner of the task bar) and select the Stop Expansive option.

To restart Expansive, select the Start Expansive option.

Alternatively, you can run exp manually in a command window.:

cd /etc/exp
exp

Expansive is configured by default to listen on port 7777. To test that it is serving pages, point your browser at http://localhost:7777. To tailor the Embedthis Expansive configuration, including the default port, you may wish to edit the exp.conf configuration file.

To get all the options for Expansive, read the man page via:

man exp
or get the command help via: 
exp --help
