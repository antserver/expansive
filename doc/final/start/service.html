Running Expansive as a System Service or Daemon

After installation, Expansive is started automatically as a system service or daemon. You can change this behavior and only start Expansive manually if you wish.

This page gives generic and windows -specific information about starting and stopping the Expansive service.

  • Generic Instructions
  • Windows Instructions

Generic Instructions

Expansive includes a manager program called appman. This program is used to configure the running of Expansive as service. Appman provides a generic way to install/uninstall, enable/disable, start/stop, and run the Expansive service regardless of the Operating System.

The Appman command takes the form:

appman commands...

Where commands can be:

CommandDetails
installInstall Expansive service
uninstallUninstall Expansive service
enableEnable Expansive to start automatically on system boot
disableDisable Expansive from automatically starting
startStart Expansive manually
stopStop Expansive
runRun Expansive and monitor its execution. Restart if it fails.

Stopping Expansive as a Service

Use the following command to stop Expansive running as a service permanently:

appman disable

Run the following commands to restore Expansive running as a service:

appman enable
appman start

Windows Instructions

On Windows, Expansive is managed by two programs:

  • The Expansive Manager
  • The Expansive Monitor

The Appman manager program is installed as a windows service that starts, manages and restarts Expansive. If Expansive fails, Appman will automatically restart it.

Windows services run in the background with little interaction with the user. So to provide more convenient access to manage Expansive, the Expansive Monitor program runs and is accessible from the windows system tray (taskbar notification area). You can start and stop Expansive using either the manager or Monitor programs.

Appman — Expansive Manager Service

You can interface with the Appman program via the Windows Services Applet. To invoke the applet select:

Start -> Control Panel -> Administrative Tools -> Services

Then select and double click on the Embedthis Expansive service. A dialog panel will be displayed from which you can start and stop the manager (appman) process.

When started or stopped, the manager will first do the same operation on the real Expansive instance. Pausing the manager will have no effect on Expansive, but will pause the manager itself.

Once installed, you will need to start the service via the Services control applet.

Expansive Monitor

The Windows Expansive Monitor is launched when you log in. It a quick and easy way to start, stop, and restart Expansive. You can also launch a browser to display the documentation or the Expansive management console (future feature).

When you select 'stop' Expansive, you actually stop the manager process which in turn stops Expansive. Similarly, when you select 'start' Expansive, you are really starting the manager which detects that Expansive is not running and automatically launches it.
