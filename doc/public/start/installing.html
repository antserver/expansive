Installing Expansive

Please read the overview information below, then refer to the specific instructions for your operating system. The Expansive binary installation is available for:

  • Mac OS X Installation Instructions
  • Windows Installation Instructions
  • Linux Installation Instructions

Extension Packages

Expansive uses the Pak manager to install and manage extension plugins and packages such as Markdown, Less or Sass. Browse the online Pak Catalog for extension packages or visit the Pak Web Site.

Source Code

The Expansive source code is provided as a separate download distribution. It is also available via the source code repository for developers who wish to participate in the development of Expansive.
