Installing on MAC OS X

The Embedthis Expansive install distribution for Mac OS X is published as a an installable pkg file.

Installing the PKG Image

  1. Double click on the PKG image.

Removing Expansive

  1. Run a terminal and change directory to the installation directory for Expansive.
  2. Run the remove script in the install directory for Expansive by typing:
sudo ./uninstall

You can run exp via:

cd /etc/exp
exp

To get all the options for Expansive read the man page via:

man exp
or get the command help via: 
exp --help
