Expansive Quick Start

  1. Download

    First download Expansive on your system. You can download a binary package from https://embedthis.com/exp/download.html.

  2. Installing Expansive

    Install Expansive on your system by following the instructions relevant for your system.

  3. Quick Tour

    Take the Expansive Tour. This will give you a quick overview of Expansive and how to use it.

  4. Read the doc

    There is a wealth of documentation to read. Try the User's Guide and Developer's Guide first.
