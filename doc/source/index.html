Embedthis Expansive™ Overview

Expansive is a fast and flexible static web site generator. It quickly creates HTML pages from layouts, partials and contents pages. Expansive brings dynamic content to static web sites and powerful templating to all.

Why Expansive?

There are two key reasons we created Expansive.

  • There are many web site site generators, so why create another one? The problem is that most existing site generators are slow, very slow, often painfully slow. When creating or managing a web site, you need tools that do not delay the creative design process. They need to be as fast as possible. Expansive is designed to be fast, without sacrificing features.
  • The second reason is that existing site generators frequently provide poor diagnostics. This may be fine when everything works, but when a page fails to render due to a scripting error -- debugging can be a nightmare.

Expansive addresses these two flaws: Expansive is very fast to generate and update site contents, and if you have made a script error, it shows exactly where the error is.

That was the genesis, however, Expansive has now gone beyond its original goals and now offers a powerful, flexible environment for creating web sites.

Extensive Features

Expansive will generate static web sites from pages, layouts, partials and scripts. It has the following major features:

  • Ejscript templates for dynamic content
  • Layout pages to define the look and feel of site
  • Partial pages to include replicated content
  • Per-page meta data to control layouts and partials
  • Integrated web server to serve content
  • Watch for changes and automatically render site
  • Scripting support via change events
  • Automatic sitemap creation
  • Content transition pipeline
  • Open plugin architecture

Nice Parts

While Expansive has a strong feature set, there are some characteristics that make Expansive particularly sweet.

  • Fast, fast
  • Power of Ejscript in every page
  • Simple code. By leveraging Ejscript, the code is simple and clean
  • Easy to create and install new plugins
  • Great diagnostics when things go wrong
